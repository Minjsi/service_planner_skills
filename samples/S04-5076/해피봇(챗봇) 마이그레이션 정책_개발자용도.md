## 용어 매핑

| DB 컬럼 | 개발 용어 | 기획 용어 |
|---------|----------|----------|
| `per_price_max_traffic` | 건당 과금 최대 제한 트래픽 | 건당과금 사용유무 |
| `per_price_use` | 건당 과금 사용 여부 | 건당 과금 사용 여부 |

---

## 무제한 트래픽 설정 조건

`site_cb_plan_history` 테이블 기준:

| 조건 | 값 | 비고 |
|------|-----|------|
| `per_price_max_traffic` | 0 | 제한 없음 |
| `per_price_use` | 1 (USE) | 건당 과금 사용 |

**두 조건이 모두 충족되어야 무제한 트래픽.**

> **유의:** `per_price_use` 컬럼은 `site_cb_plan`과 `site_cb_plan_history` 두 테이블에 모두 존재하며 항상 같은 값을 유지해야 합니다.

## 마이그레이션 정책 (price_code별)

### 1. 개별 요금제 (PC_xxx)

**정책:** 기존 챗봇 요금 그대로 유지. 마이그레이션 시 변경하지 않음.

| 항목 | 값 |
|------|-----|
| `per_price_use` | 기존 값 유지 |
| `per_price_max_traffic` | 기존 값 유지 |
| `site_cb_plan` | 기존 row 그대로 사용 |

---

### 2. TF-000 (해피봇 Free)

**정책:** 건당 과금 미사용.

| 항목 | 값 |
|------|-----|
| `per_price_use` | 0 (UNUSE) |
| `per_price_max_traffic` | 0 |

---

### 3. PS-000 (해피봇 Plus)

**정책:** `ht_plan_product_mapping` 기반 V2 챗봇 플랜 생성. 무제한 트래픽 설정.

| 항목 | 값 |
|------|-----|
| `per_price_use` | 1 (USE) |
| `per_price_max_traffic` | 0 (무제한) |

> `site_cb_plan` + `site_cb_plan_history` row 신규 생성

---

### 4. TC-000 / TC-001 / TC-002 / TC-003 (해피봇 1,000 / 3,000 / 10,000 / 40,000)

**정책:** `ht_plan_product_mapping` 기반 V2 챗봇 플랜 생성. 기존 `per_price_use` 값에 따라 분기.

#### Case A: 기존 `per_price_use = 1` (건당 과금 ON)

| 항목 | 값 | 근거 |
|------|-----|------|
| `per_price_use` | 1 (USE) | 기존 값 유지 |
| `per_price_max_traffic` | **0 (무제한)** | 건당 과금 ON → 무제한 트래픽 |

#### Case B: 기존 `per_price_use = 0` (건당 과금 OFF)

| 항목 | 값 | 근거 |
|------|-----|------|
| `per_price_use` | 0 (UNUSE) | 기존 값 유지 |
| `per_price_max_traffic` | **현재 플랜의 max_traffic** | 기존 제한 트래픽을 초과 제한으로 설정 |

> `max_traffic` 참조 값:

| price_code | 요금제명 | max_traffic |
|-----------|---------|-------------|
| TC-000 | 해피봇 1,000 | 1,000 |
| TC-001 | 해피봇 3,000 | 3,000 |
| TC-002 | 해피봇 10,000 | 10,000 |
| TC-003 | 해피봇 40,000 | 40,000 |

## 요약 테이블

| price_code | 요금제명 | per_price_use | per_price_max_traffic |
|-----------|---------|--------------|----------------------|
| PC_xxx | 개별 요금제 | 기존 유지 | 기존 유지 |
| TF-000 | 해피봇 Free | 0 (UNUSE) | 0 |
| PS-000 | 해피봇 Plus | 1 (USE) | 0 (무제한) |
| TC-xxx (per_price_use=1) | 해피봇 1,000~40,000 | 1 (USE) | 0 (무제한) |
| TC-xxx (per_price_use=0) | 해피봇 1,000~40,000 | 0 (UNUSE) | max_traffic 값 (1,000~40,000) |

---

## 관련 테이블

| 테이블 | 역할 |
|--------|------|
| `site_cb_plan` | 현재 적용 중인 챗봇 플랜 (`per_price_use` 포함) |
| `site_cb_plan_history` | 챗봇 플랜 이력 (`per_price_use` + `per_price_max_traffic` 포함) |
| `ht_plan_product_mapping` | 플랜별 챗봇 상품 매핑 (V2 product_ref_id → price_code) |
| `public_price_history` | 공용 요금제 가격/트래픽 정보 (max_traffic 참조용) |
